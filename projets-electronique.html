Une horloge fait avec Arduino


  • Ce projet est une horloge fait avec Arduino qui dispose de plusieurs fonctionnalités intéressantes.

    Il est possible d'afficher l'heure actuelle, la date, la température et le taux d'humidité. L'horloge peut être utilisée la nuit mais est extrêmement lumineuse (et je sais de quoi je parle...).

  • Langage utilisé : Python

  • Nombre de lignes de code : 113

  • État du projet : Actuellement terminé ✅ -- Version 2.0
